# 一个日志引发的性能问题


```
import os
import time

fp = open('test.txt', mode='w')

start_time = time.time()
for _ in range(100000):
	fp.write('a')
	fp.flush()
	os.stat('test.txt')
end_time = time.time()
print(f'Used time: {end_time - start_time}')
```
